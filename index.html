HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements
  • Semantic HTML elements plays an important role in making sure your webpage is accessible to the widest variety of users including bthose who use assistive technologies such as screen readers
  • The src attribute defines the loaction or source of the image file
  • The alt attribute contains a text string that describes the appearance and functionality of the image.
  • Elements refers to the entire element from opening tag to closing tag whereas tag refers only to what appears inside the angle brackets
  • Head elements h1,h2,h3,h4,h5 and h6 represents the level of heading a given text block represents with texts in the h1 element being particularly large

CSS

  • css is used to style HTML elements which allows for creation of not only well-structured but also visually appealing website
  • 3 ways to style a webpage are inline css, internal css style sheet and external css style sheet
  • Inline css are used for quick and permanent changes and are less flexible than internal and external because each inline style must be edited individually whenever you make a design change.
  • External css style sheet is the most common and preferred way as it allows developers to keep all their css rules in a seperate file which makes design changes easier.
  • A margin indicates how much space we want around the outside of an element
  • A Padding indicates how much space we want around the content inside an element
  • Class attributes allows us to share a css rule to any element we choose by assigning the rule to a class attribute with a class selector.The class attribute can then be assigned to the element in the HTML enabling us to freely chosse which elements will be assigned the css rule.

Git

  • Git checkout -b branchname: creates new feature branch and automatically switches to it
  • Git add -A: commits all files in the working branch
  • Git commit -m descriptive message: shows implemented changes made
