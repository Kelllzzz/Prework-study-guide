HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements
  • Semantic HTML elements plays an important role in making sure your webpage is accessible to the widest variety of users including bthose who use assistive technologies such as screen readers
  • The src attribute defines the loaction or source of the image file
  • The alt attribute contains a text string that describes the appearance and functionality of the image.
  • Elements refers to the entire element from opening tag to closing tag whereas tag refers only to what appears inside the angle brackets
  • Head elements h1,h2,h3,h4,h5 and h6 represents the level of heading a given text block represents with texts in the h1 element being particularly large

CSS

  • css is used to style HTML elements which allows for creation of not only well-structured but also visually appealing website
  • 3 ways to style a webpage are inline css, internal css style sheet and external css style sheet
  • Inline css are used for quick and permanent changes and are less flexible than internal and external because each inline style must be edited individually whenever you make a design change.
  • External css style sheet is the most common and preferred way as it allows developers to keep all their css rules in a seperate file which makes design changes easier.
  • A margin indicates how much space we want around the outside of an element
  • A Padding indicates how much space we want around the content inside an element
  • Class attributes allows us to share a css rule to any element we choose by assigning the rule to a class attribute with a class selector.The class attribute can then be assigned to the element in the HTML enabling us to freely choose which elements will be assigned the css rule.

Git

  • Git status: checks what branch you are currently on
  • Git checkout main: Switches to the main branch
  • Git pull origin main: Pull down the latest main branch
  • Git checkout -b branchname: creates new feature branch and automatically switches to it
  • Git add -A: commits all files in the working branch
  • Git commit -m descriptive message: shows implemented changes made

JavaScript

  • A variable,one of the core feature of javascript, is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Using conditional statements as 'if' statements allows us to interrupt control flow.
  • Arrays are used to hold data that is related in some way.
  • The first item in an array has an index of 0 not 1.The indices or location identifiers will alweays be whole numbers that increase by 1 each time.
  • A 'for' loop contains 3 important statements; The First determines the STARTING POINT for our loop, the second statement is the CONDITION - as long as the number in the variable is less than the length of the array, the loop will keep running. The final statement is what allows the array iterate over each item.
  • Placing /* Before and */ After the block of code is known as Commenting out a code. This makes the block of code inactive without deleting it
  • A function is a set of instructions that tells the computer how to perform a certain task.They are reusable blocks of code that contains instructions to perform a specific task.
